# 💧 喝水提醒助手

一款简约的WPF桌面喝水提醒应用，帮助您养成健康的饮水习惯。

## 功能特色

### 🎯 核心功能
- **每日饮水记录** - 准确记录每次饮水量，本地存储数据不丢失
- **智能提醒** - 可自定义提醒间隔，在设定时间内定时提醒
- **进度追踪** - 实时显示今日饮水进度和完成百分比
- **快速记录** - 预设常用水量(250ml/500ml/1000ml)，一键添加
- **自定义设置** - 可设置每日目标水量和提醒间隔

### ✨ 贴心设计
- **每日美句** - 每天显示不同的励志美句，治愈心灵
- **进度可视化** - 直观的进度条显示当日完成情况
- **鼓励反馈** - 达到不同进度时给予鼓励提示
- **系统托盘** - 最小化到系统托盘，右键菜单快速操作
- **托盘提醒** - 气泡提示，不打扰正常工作
- **简约界面** - 清新简洁的UI设计，使用体验舒适

## 使用说明

### 基本操作
1. **记录饮水** - 点击快速按钮(250ml/500ml/1000ml)或输入自定义水量
2. **查看进度** - 主界面显示今日饮水总量和目标完成百分比
3. **管理记录** - 在今日记录列表中可以删除错误记录
4. **清空记录** - 点击"清空"按钮可清除当日所有记录
5. **系统托盘** - 关闭或最小化窗口时程序保持在系统托盘运行
6. **托盘操作** - 右键托盘图标可快速添加饮水记录或显示主窗口

### 设置配置
1. **每日目标** - 设置每日饮水目标(默认2000ml)
2. **提醒间隔** - 设置提醒间隔时间(默认60分钟)
3. **开启提醒** - 点击"开启提醒"按钮启动定时提醒功能

### 数据存储
- 所有数据自动保存到本地 `%AppData%\WaterReminder` 目录
- `records.json` - 存储所有饮水记录
- `settings.json` - 存储用户设置


## 技术栈

- **框架**: WPF (.NET 8.0)
- **UI**: XAML + 自定义样式
- **数据存储**: JSON文件 (Newtonsoft.Json)
- **定时器**: DispatcherTimer
